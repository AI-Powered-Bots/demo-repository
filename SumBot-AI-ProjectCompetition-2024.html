üíª Installation

To get started with SumBot FreeCode, follow these steps:

  1. Clone the repository: git clone https://github.com/Mindful-AI-Assistants/SumBot-FreeCode.git
  2. Install the required dependencies (Optional):
    • Create a requirements.txt file with necessary packages, or run pip install openai slack-sdk in your virtual environment.
    • Set up your environment variables to connect with OpenAI and Slack APIs.
  3. Run the application:
    • Use the provided run.py script to start the bot, or integrate it into your automation workflows via Zapier.
